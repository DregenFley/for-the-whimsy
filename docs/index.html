About the pack

A world worth wandering

For the Whimsy is a carefully curated survival modpack built around one simple idea: the journey should matter. Without elytras, the world feels vast again. You'll build roads, share boats, and actually care about your neighbours.

It's cosy without being easy, whimsical without being silly. Whether you're settling down in a mossy cottage or striking out on horseback to find a new village, there's always something worth exploring just over the next hill.

Getting started

Installing in four steps

1

Download CurseForge if you don't have it already — it manages your modpack profile and launches the game.

2

Download the modpack using the button above. In CurseForge, go to Minecraft → Create Custom Profile → Import and select the downloaded zip. CurseForge will install all the mods automatically.

3

Download the setup tool and run it. Windows may show a security warning — click "More info" then "Run anyway". The tool is open source so you can verify it yourself.

4

Copy your data across — pick your old profile as the source and For the Whimsy as the destination. The tool will carry over your map, keybinds, and settings.
